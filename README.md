# moysklad-shop-bot

Бот для розничного магазина. Клиент хотел чтобы покупатели могли сами проверить наличие товара без звонков.

Данные берутся из МойСклад в реальном времени, остатки показываются по каждой точке отдельно.

## Что есть

- верификация 18+ (специфика клиента)
- поиск по названию товара
- остатки по складам
- пагинация если результатов много
- адреса точек с яндекс картами
- контакт менеджера

## Стек

aiogram 3, aiohttp, python-dotenv

## Запуск

```
pip install -r requirements.txt
cp .env.example .env
python main.py
```

заполни .env своими токенами перед запуском
