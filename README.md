# Telegram-бот для розничного магазина + МойСклад

Бот сделан для небольшой розничной сети. Основная задача — дать покупателям возможность самим проверить наличие товара, не звоня в магазин.

## Что умеет

- верификация 18+ при первом запуске
- поиск товара по названию — данные тянутся из МойСклад в реальном времени
- показывает остаток по каждой точке отдельно
- пагинация если результатов много
- адреса магазинов с ссылкой на карту
- кнопка "написать менеджеру"

## Стек

- Python 3.11
- aiogram 3, FSM
- aiohttp — асинхронные запросы к МойСклад API
- python-dotenv

## Запуск

```bash
pip install -r requirements.txt
cp .env.example .env
# заполни .env своими токенами
python main.py
```

## Структура

```
├── main.py          # точка входа, polling
├── handlers.py      # роутер, FSM-хендлеры
├── moysklad.py      # работа с МойСклад API
├── config.py        # настройки, список магазинов
└── requirements.txt
```

## Настройка

В `config.py` прописываются адреса точек и лимиты поиска. Токены — через `.env`.
